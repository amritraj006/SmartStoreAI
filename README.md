# SmartStore AI

AI-powered E-commerce Admin Dashboard using the MERN Stack.

---

# Tech Stack

## Frontend
- React.js
- Tailwind CSS
- React Router
- Axios
- Recharts

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

---

# Project Structure

```bash
smartstore-ai/
│
├── client/         # React Frontend
├── server/         # Node.js Backend
│
├── README.md
└── .gitignore
```

---

# Installation

## Clone Repository

```bash
git clone <repo-url>
cd smartstore-ai
```

## Install Frontend Dependencies

```bash
cd client
npm install
```

## Install Backend Dependencies

```bash
cd ../server
npm install
```

---

# Run Frontend

```bash
cd client
npm run dev
```

---

# Run Backend

```bash
cd server
npm run dev
```

---

# Environment Variables

Create `.env` inside `server/`

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
